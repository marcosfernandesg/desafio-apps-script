# Sistema de Processos - Google Apps Script

Sistema web desenvolvido com Google Apps Script para controle de processos bancários.

## 🔗 Acesso ao sistema
O sistema já está publicado como Web App e pode ser acessado diretamente pelo link:

👉 https://script.google.com/macros/s/AKfycbx09s9a9CoK2LlV90kVIfV_Xkp1MsU4gmy4LXVzXV3Dq5L4E4phD2hfwcKDEIr0ijfU/exec

## 📊 Base de dados
O sistema utiliza uma planilha Google Sheets como banco de dados.

## ⚙️ Funcionalidades
- Listagem de processos
- Filtro por cliente, produto e status
- Criação de novos processos
- Edição de processos
- Exclusão de processos

## 🛠️ Tecnologias utilizadas
- Google Apps Script
- HTML + Tailwind CSS
- Google Sheets

## 📁 Estrutura do projeto
- `Servidor.gs` → lógica backend (Apps Script)
- `Cliente.html` → interface do sistema

## 📌 Observação
O sistema é totalmente web, não sendo necessário instalação. Basta acessar o link do Web App.
